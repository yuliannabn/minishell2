# minishell2

Shell based on tcsh, creates a binary tree as a way to parse commands (that way, adding any other operator is easy to do).

## FEATURES

* | (pipes)
* ; (separator)
* '>>' and '<<' (heredoc)
* '<'  and  '>'
* '&&' and  '||'
* cd
* setenv
* unsetenv
* run binary (with PATH support)

## COMPILATION
> make re
